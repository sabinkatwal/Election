# Nepal Election 2082 Live Dashboard

This is a web-based live dashboard for the Nepal Election 2082 (2026 AD), providing real-time visualizations and insights into electoral data, focusing on proportional representation analysis.

## Features

- Real-time election data updates
- Interactive charts and graphs powered by Chart.js
- Dark and light theme support
- Responsive design for various devices
- Multilingual support (Nepali and English)

## Technologies Used

- HTML5
- CSS3 (with custom properties for theming)
- JavaScript
- Chart.js for data visualization

## How to Use

1. Open `nepal_dashboard_v7.html` in a modern web browser (Chrome, Firefox, Safari, etc.).
2. The dashboard will load with the latest election data.
3. Use the theme toggle to switch between dark and light modes.
4. Navigate through different sections to view party standings, vote distributions, and more.

## Requirements

- A modern web browser with JavaScript enabled
- Internet connection for live data updates (if applicable)

## Troubleshooting

- If the dashboard doesn't load, ensure JavaScript is enabled in your browser.
- For any issues, check the browser console for error messages.
- If charts don't display, verify that Chart.js is loading from the CDN.
